Ungkapan untuk menyemangati

Langkah kecil hari ini akan membawamu ke pencapaian besar di masa depan,
Tetaplah berjuang, karena kerja kerasmu hari ini akan jadi hadiah manis esok hari
Semangat itu bukan tentang seberapa kuat kita,
tapi seberapa teguh kita bertahan saat ingin menyerah
Jangan berhenti sebelum kamu bangga dengan hasilmu sendiri.
